# todo-react-app
This is todo frontend application in React + Vite and Material UI

This application implements Todo app mentioned over https://todomvc.com/ 

## Steps to run on local
1. Clone the project
2. Go to Project root folder
3. Run "npm install"
4. Run "npm run dev"
